## 🛠️ Stack Tecnológico

- **Frontend:** React 18 + Styled Components
- **Autenticación:** Firebase Auth
- **Base de datos:** Firestore (NoSQL)
- **API de contenido:** TMDB API
- **Navegación:** React Router
- **Reproductor:** React YouTube

---

## ⚙️ Instalación

```bash
git clone https://github.com/[usuario]/churo-chapaco.git
cd churo-chapaco
npm install
npm start
```
